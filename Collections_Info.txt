Collecties en velden:

- Configuration (http://localhost:3000/collectionapi/configuration)
    - roles: Object Array
        - name: String;
        - allowedlinks: String Array;
    - server: Object
        - triggeraddress: String;

- Meteor.users (http://localhost:3000/collectionapi/users)
    - createdAt: Date;
    - profile: Object
        - name: String;
        - email: String;
        - verified: boolean;
    - roles: String Array;
    - emails: Array             (Wordt alleen weergeven bij een Standaard account)
        - address: String;
        - verified: boolean;    (Deze variable wordt niet gebruikt in dit project)
    - services: Object
        - linkedin: Object;     (Wordt alleen weergeven bij een Linkedin account)
        - facebook: Object;     (Wordt alleen weergeven bij een Facebook account)
        - twitter: Object;      (Wordt alleen weergeven bij een Twitter account)
        - password: Object;     (Wordt alleen weergeven bij een Standaard account)
        - resume: Object;
NOTE: Voor meer informatie wat er in de linkedin, facebook, twitter, emails of password objecten staat kunt u de volgende commando gebruiken: Meteor.users.find()
